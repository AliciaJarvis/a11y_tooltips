Native Tooltip

ARIA 1.1 Tooltip specification

Tool tips should be able to be activated by:

  • hover, after a short pause.
  • on focus of an element where there is no press or click listener (e.g. a form control, or button or link which perform other actions on keypress or click).

A user name can either be your phone number
